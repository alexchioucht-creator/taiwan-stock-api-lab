\# 🇹🇼 Taiwan Stock API MVP Lab



這是一個利用 \*\*臺灣證券交易所 (TWSE) OpenAPI\*\* 進行 API 介接的最小可行性產品 (MVP) 實驗。



\## 🎯 實驗目的

1\. 驗證免驗證機制（No-Auth）的政府 Open API 連通性。

2\. 實作非同步請求 (`async/await`) 與錯誤處理。

3\. 進行大數據量 JSON 陣列的線上篩選與資料清洗。
